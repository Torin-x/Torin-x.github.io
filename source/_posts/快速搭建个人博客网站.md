## 0 前言

分享我如何在三天内搭建的个人简易博客网站，有句话说的好：前人造的轮子我不用那我不成傻子了。

当然如此快速的搭建是基于 ([Hexo](https://hexo.io/zh-cn/)) 这个成熟的框架，下面介绍如何使用。



## 1.安装Hexo

#### 1.1  安装Git 和 Node.js (准备工作)

 详细内容在官方文档和各种教程都能搜到，不多阐述

[Git - Downloads](https://git-scm.com/downloads)

[Node.js](https://nodejs.org/zh-cn)

#### 1.2 安装 Hexo

新建文件夹作为博客根目录，进入目录在终端中输入

```powershell
npm install -g hexo-cli
```



## 2.建立网站

同样在根目录下

```powershell
hexo init "folder"
```

其中 "folder" 为你网页想取的名字

```powershell
cd "folder"
npm install
```

如果上述 npm 安装出现 **error** 则大概率是权限问题，请用 **管理员身份** 打开终端进入该文件重新安装



初始化完成可以看到如下文件夹

```bash
.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes
```

这样我们就得到了一个最简单的网页

执行下面命令启动

```powershell
hexo s
```

弹出链接 http://localhost:4000/ ，在浏览器打开即可看到初始网页。

更多 hexo 使用方式请看官方文档 [指令 | Hexo](https://hexo.io/zh-cn/docs/commands)



## 3.安装主题

#### 3.1 安装

Hexo 官网给出了自定义板块，其中就包括了 [Themes | Hexo](https://hexo.io/themes/) ，在这里面有大量个人上传的主题，挑一个你喜欢的用吧。

点进链接能进入到 GitHub 项目

将项目克隆到本地

```powershell
cd themes
git clone "git_url"
```

其中 "git_url" 为主题的 GitHub 项目链接。



这时打开根目录下的 _config.yml 将里面的

```yaml
theme : "主题文件夹的名字"
```

更多 _config.yml 配置请看作者 GitHub 项目下的具体配置说明。



#### 3.2  运行

```powershell
hexo s
```

刷新网页即可显示。

## 5.将网站从本地部署改成 GitHub 部署

#### 5.1  新建 GitHub 仓库

 首先新建仓库  [New repository](https://github.com/new)  填入 "user".github.io , "user" 为你GitHub的**用户名** 。

#### 5.2获取 SSH 密钥

将邮箱换成你自己的

```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
```

复制 ssh密钥

```powershell
Get-Content $env:USERPROFILE\.ssh\id_ed25519.pub | Set-Clipboard
```

#### 5.3 **添加到 GitHub：**

- 回到 GitHub 网站，点击右上角的头像，选择 **"Settings"**。
- 在左侧菜单中，点击 **"SSH and GPG keys"**。
- 点击绿色的 **"New SSH key"** 按钮。
- **Title** 可以随意填写。
- 在 **Key** 的文本框中，粘贴密钥。
- 点击 **"Add SSH key"**。

#### 5.4 测试

```bash
ssh -T git@github.com
```

如果是第一次链接会弹出提示，此时输入 **yes** 即可

出现

```bash
Hi <Your-Username>! You've successfully authenticated, but GitHub does not provide shell access.
```

代表连接成功。

#### 5.5  部署hexo

安装 hexo 插件来将文件推送到 GItHub

**进入博客根目录**

``` bash
npm install hexo-deployer-git --save
```



**配置 _config.yml** 

```yaml
deploy:
  type: git
  repo: git@github.com:你的GitHub用户名/你的GitHub用户名.github.io.git
  branch: main
```



**启动**

```bash
hexo cl
hexo d -g
```

此时回到仓库发现已经不是空的，稍等几分钟访问  https://user.github.io/ 可以看到你的网站，即代表部署完成。每次做修改完按上述 **启动** 即可修改。
